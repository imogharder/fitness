About This Project

Body Stats Tracker started as a minimal body measurement logger and is evolving one version at a time into a full fitness platform. Each release is self-contained, fully functional, and live — so you can explore the exact state of the app at any point in its development.

Built entirely with plain HTML, CSS and JavaScript. No frameworks, no databases, no cloud — just clean code that runs entirely in the browser using localStorage.
